Kao studentu bitna mi je informacija koliko novca u nekom periodu potrošim na hranu u menzi. Portal issp.srce.hr na svojim stranicama nudi mogućnost pregleda svih izdanih računa, no ne nudi konkretan izračun potrošenog novca (nego samo cijenu bez subvencije i koliki dio je subvencioniran). Ova skripta omogućuje jednostavno rješenje tom problemu.…

**** 1. prijavi se s AAI@eduhr identitetom na: https://issp.srce.hr/Account/OdaberiPrijavu?ReturnUrl=%2Fstudent%2Fstudentracuni
**** 2. odi na bilo koji ispis racuna - opcija ispod profilne slike (prikaz zadnjih 30 dana na webu ili pdf sa svim racunima)
**** 3. ctrl + A (sve oznaci)	ctrl + C (kopiraj)	ctrl + V (zalijepi u folder "myData.txt)
**** 4. promijeni JEDNU liniju koda u menza.py skripti: linija 32. unesi mjesec i godinu za koji
zelis izracun
**** 5. trebas python: https://www.python.org/downloads
**** 6. ako trebaš, guglaj "how to run a python script"
**** 7. Pokreni menza.py python skriptu i uzivaj!
